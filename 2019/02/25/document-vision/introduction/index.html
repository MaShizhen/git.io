计算机视觉介绍

计算机视觉结合了计算机科学的许多领域，从（低级）图像数据中提取（高级）信息。

管道

许多计算机视觉系统使用管道模式。 此管道通常可以分解为以下步骤：

Data => Preprocessing -> Segmentation -> Feature Extraction -> Classification => Information

数据 =>预处理 ->分段 ->特征提取 ->分类 =>信息

预处理

预处理图像时，通常应用去除噪声，热像素等。将其视为使用GIMP和G’MIC准备图像。 这一切都是为了让后续算法的生活更简单。

分段

正式地说，分段是将每个像素分配给一个组的过程。 使用文本时，这通常是“文本是黑色的”和“纸是白色的” - 这也称为二值化。

在处理复杂图像时，通常会提取两个以上的段 - 这通常使用区域增长和统计区域合并等算法来完成。

特征提取

在特征提取期间，诸如线条，形状，运动等的特征从单个分割图像或一系列分割图像中提取（注意：这里是龙！）。 举一些特别流行的算法：Canny边缘检测器和Hough变换可用于提取边缘和线条。

分类

在分类过程中，所有提取的特征都被分类为有用的信息，例如“这是一个置信度为90％的’X’或置信度为60％的’x’”。 此步骤通常涉及某种决策树和大量统计信息，有时还涉及机器学习。

既然我们谈论现实世界的图像和现实世界是棘手的，那么有一种信心是很好的。 将其视为保险或信任因素，可防止您的申请遭受致命的误解。
