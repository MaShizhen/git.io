DocumentVision

DocumentVision 是一个用于处理并理解扫描文档的nodejs库。

特点

  • 使用jpeg-compressor, LodePNG和像素缓冲区加载图片
  • 使用Leptonica（版本1.69）和OpenCV（版本2.4.9）进行图像处理
  • 使用LSWMS进行线段检测
  • OCR使用Tesseract（3.03版）
  • 使用ZXing的OMR for Barcodes（版本2.3.0）
